# Contributing

Thanks for helping improve this FCL list.

## Add a Paper

Please use this format:

```markdown
- Paper Title (**Venue Year**) [[paper](link)] [[pdf](link)] [[code](link)] <sub><img alt="tags: FCIL, replay, resource" src="https://img.shields.io/static/v1?label=tags&amp;message=FCIL%20%7C%20replay%20%7C%20resource&amp;color=1f6feb&amp;style=flat-square" height="16"></sub>
```

If a PDF or code link is unavailable, omit that tag.

## Preferred Links

- Prefer official DOI, arXiv, OpenReview, CVF, PMLR, NeurIPS, ACM, IEEE, Springer, Elsevier, MDPI, journal, or author pages.
- Prefer official code repositories linked by the paper or authors.
- Do not add unofficial PDF mirrors.
- Do not commit PDF files to this repository.

## Sorting

- Sort papers by year in descending order.
- Inside each year, keep entries roughly alphabetical by title unless a maintainer chooses a stronger venue-first ordering.

## Metadata

When possible, update both:

- `README.md`
- `papers.bib`

## Taxonomy

In `README.md`, use one tiny colored Shields badge at the end of an item, for example `<sub><img alt="tags: FCIL, replay, resource" src="https://img.shields.io/static/v1?label=tags&amp;message=FCIL%20%7C%20replay%20%7C%20resource&amp;color=1f6feb&amp;style=flat-square" height="16"></sub>`.
