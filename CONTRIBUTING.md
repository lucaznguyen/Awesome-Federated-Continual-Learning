# Contributing

Thanks for helping improve this FCL list.

## Add a Paper

Please use this format:

```markdown
- Paper Title (**Venue Year**) [[paper](link)] [[pdf](link)] [[code](link)] <sub>compact tags</sub>
```

If a PDF or code link is unavailable, omit that tag.

## Preferred Links

- Prefer official DOI, arXiv, OpenReview, CVF, PMLR, NeurIPS, ACM, IEEE, Springer, Elsevier, MDPI, journal, or author pages.
- Prefer official code repositories linked by the paper or authors.
- Do not add unofficial PDF mirrors.
- Do not commit PDF files to this repository.

## Sorting

- Sort papers by year in descending order.
- Inside each year, keep entries roughly alphabetical by title unless a maintainer chooses a stronger venue-first ordering.

## Metadata

When possible, update both:

- `README.md`
- `papers.bib`

## Taxonomy

In `README.md`, use only tiny compact tags at the end of an item, for example `<sub>FCIL &middot; replay &middot; resource</sub>`.
